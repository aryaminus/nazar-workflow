Copy the Copy the five files generated by the conversion to slim\satellite\data from step: 01_Data prepare

Modify the slim\datasets\satellite.py file:

    FILE_PATTERN = 'satellite %s_*.tfrecord' (tf-record filename format)
    SPLITS_TO_SIZES = {'train': 16, 'validation': 4} (Total number of training set and test set files)
    _NUM_CLASSES = 2 (Total Category Category)
    'image/format': tf.FixedLenFeature((), tf.string, default_value='jpg') (picture format, here is jpg)

Download Pre-training model Inception V3:

    http://download.tensorflow.org/models/inception_v3_2016_08_28.tar.gz

        After decompression, copy to slim\satellite\pretrained

    or,
        download http://download.tensorflow.org/models/inception_v4_2016_09_09.tar.gz
        
        and decompress and save the checkpoint to slim\satellite\pretrained

Run the following command in the slim/ folder to train:

    ```
    python train_image_classifier.py \
    --train_dir=satellite/train_dir \
    --dataset_name=satellite \
    --dataset_split_name=train \
    --dataset_dir=satellite/data \
    --model_name=inception_v3 \
    --checkpoint_path=satellite/pretrained/inception_v3.ckpt \
    --checkpoint_exclude_scopes=InceptionV3/Logits,InceptionV3/AuxLogits \
    --trainable_scopes=InceptionV3/Logits,InceptionV3/AuxLogits \
    --max_number_of_steps=100000 \
    --batch_size=32 \
    --learning_rate=0.001 \
    --learning_rate_decay_type=fixed \
    --save_interval_secs=300 \
    --save_summaries_secs=2 \
    --log_every_n_steps=10 \
    --optimizer=rmsprop \
    --weight_decay=0.00004
    ```

xecute the following command in the slim/ folder to evaluate the model's capabilities:
   
    ```
    python eval_image_classifier.py \
    --checkpoint_path=satellite/train_dir \
    --eval_dir=satellite/eval_dir \
    --dataset_name=satellite \
    --dataset_split_name=validation \
    --dataset_dir=satellite/data \
    --model_name=inception_v3
    ```

Export trained models:

    ```
    python export_inference_graph.py \
    --alsologtostderr \
    --model_name=inception_v3 \
    --output_file=satellite/inception_v3_inf_graph.pb \
    --dataset_name satellite
    ```
    in slim/ folder:

    ```
    python freeze_graph.py \
    --input_graph slim/satellite/inception_v3_inf_graph.pb \
    --input_checkpoint slim/satellite/train_dir/model.ckpt-5271 \
    --input_binary true \
    --output_node_names InceptionV3/Predictions/Reshape_1 \
    --output_graph slim/satellite/frozen_graph.pb
    ```
    in the root directory of the project (you need to change 5271 to the actual number of model training steps saved in train_dir)
